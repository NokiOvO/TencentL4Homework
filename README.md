# TencentL4Homework

实现功能：

1.游戏存档与读取、在排行榜中排名并显示；

2.一套简洁优雅的UI界面，包含主菜单、排行榜、加载界面、游戏中界面、游戏结束界面等；
使用了UMG动画来产生渐隐效果，使用RenderTarget将3D人物模型渲染到UI屏幕；
为UI按钮添加了音效，增强用户的体验。
 
3.实现了一套简单的战斗系统与小怪AI，可以自选角色加入战斗；使用虚拟摇杆和屏幕触摸检测来读取玩家移动输入，触摸UI按钮进行跑、跳、开火、装弹等动作。
